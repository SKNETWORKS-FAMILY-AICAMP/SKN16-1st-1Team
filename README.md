# 🖊 회고
<br>

 저희 팀은 이번 웹 크롤링 협업 프로젝트를 통해 수업 시간에 배운 이론을 실제로 적용해 보는 값진 경험을 할 수 있었습니다. 짧은 일정 속에서도 자료 수집부터 전처리, 모델 학습까지 ‘선택과 집중’의 중요성을 다시 한 번 깨달았고, 각 단계가 어떻게 유기적으로 연결되어야 최종 산출물이 완성되는지 전반적인 프로세스를 이해하게 되었습니다.

 코드가 막힐 때마다 조장님과 팀원들이 하나씩 순서대로 원인을 분석해 주셔서 문제를 바로잡을 수 있었고, 오류가 해결될 때마다 느낀 성취감은 협업의 진정한 즐거움을 일깨워 주었습니다. 아직 다듬어야 할 부분이 많고 더 공부할 거리도 산더미처럼 쌓여 있지만, 서로의 강점을 십분 활용하며 성장해 나가고 있다는 사실에 큰 만족을 느낍니다.

 앞으로도 이번 프로젝트에서 얻은 협업 경험과 웹 크롤링·데이터 처리 노하우를 바탕으로 더 도전적인 과제에 임하며, 함께 배우고 발전하는 팀이 되도록 노력하겠습니다.
